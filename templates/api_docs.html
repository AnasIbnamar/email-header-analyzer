01 — Analyze Headers
POST /api/analyze

Analyze raw email headers and return a full threat report with SPF/DKIM/DMARC validation, IP reputation, and risk score.

Request Body
{
  "headers": "Received: from mail.example.com...\nFrom: sender@example.com\n..."
}
Example — cURL
curl -X POST {{ request.host_url }}api/analyze \
  -H "Content-Type: application/json" \
  -d '{"headers": "From: test@example.com\nSubject: Hello"}'
Example — Python
import requests

headers = """From: attacker@evil.com
Reply-To: harvest@phish.ru
Subject: Urgent action required"""

response = requests.post(
    "{{ request.host_url }}api/analyze",
    json={"headers": headers}
)
print(response.json())

02 — Scan History
GET /api/history

Returns all previous scan results ordered by most recent first. Maximum 100 results.

Example — cURL
curl {{ request.host_url }}api/history
Response
{
  "total": 2,
  "scans": [
    {
      "id": 2,
      "scanned_at": "2026-04-25 22:12:00",
      "subject": "Your Microsoft account...",
      "from_addr": "Microsoft Account Team...",
      "risk_score": 80,
      "verdict": "HIGH RISK",
      "malicious_ips": 1,
      "warnings": 1
    }
  ]
}
03 — Get Scan by ID
GET /api/scan/:id

Returns the full report JSON for a specific scan by its ID.

Example — cURL
curl {{ request.host_url }}api/scan/1
Error Response
{ "error": "Scan not found" }
